# Setup

Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```

# Four Properties to Transform Objects(all Object3D)

- Position
- Scale
- Rotation
- Quaternion

## Position

To move objects. With **object.position** has 3 properties:

- x (left or right)
- y (upward or downward)
- z (forward or backward)

## Scale

To resize the object

## Rotation

To rotate the object

## Quaternion

To also rotate the object
